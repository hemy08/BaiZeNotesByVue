# Hemy Markdown Editor

HemyMarkdownEditor 个人的markdown编辑器项目，基于Electron + Vue+ Vite + TypeScript

具体点击工具栏帮助->关于按钮

## Project Setup

### Install

```bash
$ npm install
```

### Development

```bash
$ npm run dev
```

### Build

当前只支持Windows下的编译，其他平台未编译测试

```bash
# For windows
$ npm run build:win
```
